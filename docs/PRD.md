## 1. 产品概述

### 1.1 产品定位
布布助手是一款 macOS 桌面精灵应用，以可爱的虚拟角色形式常驻桌面，集成 AI 大语言模型能力，为用户提供便签管理、智能搜索和翻译等实用功能。

### 1.2 目标用户
- macOS 用户
- 需要桌面便签/待办管理的用户
- 需要快速查询术语、翻译文本的开发者/学生/职场人士
- 喜欢可爱桌面宠物的用户

### 1.3 核心价值
- **便捷**: 拖拽文本即可触发 AI 功能，无需切换应用
- **智能**: 集成多种主流 LLM，提供专业的解释和翻译
- **可爱**: 精美的角色形象和动画，提升使用愉悦感
- **高效**: 本地便签管理 + 定时提醒，提升工作效率

## 2. 功能需求

### 2.1 悬浮精灵系统

#### 2.1.1 角色形象
| 需求项 | 描述 |
|-------|------|
| 预设角色 | 布布（蓝色恐龙装）、伊尔（粉色恐龙装） |
| 自定义角色 | 支持用户导入自定义图片作为角色 |
| 角色切换 | 可在设置中随时切换当前使用的角色 |

#### 2.1.2 动画效果
| 状态 | 动画描述 |
|-----|---------|
| 待机 | 轻微上下浮动 + 眨眼效果 |
| 思考中 | 左右摇摆 + 头顶显示思考气泡 |
| 说话中 | 嘴巴开合动画 + 文字气泡 |
| 开心 | 跳跃动画（任务完成时触发） |
| 睡眠 | 闭眼 + Zzz 气泡（长时间无操作） |

#### 2.1.3 窗口特性
| 特性 | 描述 |
|-----|------|
| 悬浮层级 | 始终置顶，但不遮挡全屏应用 |
| 窗口拖拽 | 可拖拽到屏幕任意位置 |
| 位置记忆 | 记住上次关闭时的位置 |
| 大小调节 | 支持缩放精灵大小（50%-200%） |
| 透明度 | 可调节窗口透明度 |

#### 2.1.4 交互方式
| 交互 | 行为 |
|-----|------|
| 单击精灵 | 打开主功能面板 |
| 拖拽文本到精灵 | 弹出功能选择菜单（搜索/翻译/添加便签） |
| 右键精灵 | 显示快捷菜单 |
| 双击精灵 | 快速打开便签列表 |

### 2.3 便签管理系统

#### 2.3.1 便签属性
| 属性 | 类型 | 描述 |
|-----|------|------|
| 标题 | String | 便签标题（必填） |
| 内容 | String | 便签详细内容（可选） |
| 状态 | Enum | 待办 / 进行中 / 已完成 |
| 优先级 | Enum | 低 / 中 / 高 / 紧急 |
| 标签 | [String] | 自定义标签，支持多个 |
| 提醒时间 | Date? | 可选的提醒时间 |
| 创建时间 | Date | 自动记录 |
| 更新时间 | Date | 自动记录 |

#### 2.3.2 便签功能
| 功能 | 描述 |
|-----|------|
| 创建便签 | 支持快速创建和详细创建两种模式 |
| 编辑便签 | 修改便签所有属性 |
| 删除便签 | 支持单个删除和批量删除 |
| 状态切换 | 快速切换便签状态 |
| 搜索筛选 | 按标题、内容、标签、状态、优先级筛选 |
| 排序 | 按创建时间、优先级、状态排序 |
| 拖拽排序 | 手动拖拽调整顺序 |

#### 2.3.3 提醒功能
| 功能 | 描述 |
|-----|------|
| 设置提醒 | 为便签设置提醒时间 |
| 本地通知 | 到达提醒时间时发送系统通知 |
| 精灵提醒 | 精灵角色弹出提醒气泡 |
| 重复提醒 | 支持每天/每周/每月重复 |
| 提前提醒 | 可设置提前 5/15/30/60 分钟提醒 |

#### 2.3.4 便签视图
| 视图 | 描述 |
|-----|------|
| 列表视图 | 紧凑的列表形式 |
| 卡片视图 | 卡片式布局，显示更多信息 |
| 看板视图 | 按状态分列的看板 |
| 日历视图 | 按提醒日期展示（可选） |

### 2.4 智能搜索功能

#### 2.4.1 触发方式
1. 拖拽选中文本到精灵
2. 在搜索界面直接输入
3. 全局快捷键（可配置）

#### 2.4.2 搜索类型
| 类型 | 描述 | AI Prompt 示例 |
|-----|------|---------------|
| 术语解释 | 解释专业术语或概念 | "请用简洁的语言解释：{text}" |
| 代码解析 | 解释代码功能和逻辑 | "请解释这段代码的功能：{code}" |
| 问题解答 | 回答用户的问题 | 直接发送用户问题 |
| 总结归纳 | 总结长文本要点 | "请总结以下内容的要点：{text}" |

#### 2.4.3 结果展示
- 流式输出，实时显示 AI 回复
- 支持 Markdown 渲染
- 代码高亮显示
- 一键复制结果

#### 2.4.4 搜索历史
| 功能 | 描述 |
|-----|------|
| 自动保存 | 自动保存搜索记录 |
| 历史列表 | 查看历史搜索和结果 |
| 重新搜索 | 点击历史记录重新搜索 |
| 清除历史 | 支持单条删除或全部清除 |

## 3. 非功能需求

### 3.1 性能要求
| 指标 | 要求 |
|-----|------|
| 启动时间 | < 2 秒 |
| 内存占用 | < 100 MB（空闲状态） |
| CPU 占用 | < 1%（空闲状态） |
| 动画帧率 | 60 FPS |

### 3.2 兼容性
| 项目 | 要求 |
|-----|------|
| macOS 版本 | macOS 13.0 (Ventura) 及以上 |
| 芯片架构 | 支持 Intel 和 Apple Silicon |
| 屏幕 | 支持 Retina 和非 Retina 显示 |

### 3.3 安全性
| 项目 | 措施 |
|-----|------|
| API Key 存储 | 使用 Keychain 加密存储 |
| 数据存储 | 本地 Core Data，不上传云端 |
| 网络请求 | 仅与配置的 LLM API 通信 |

### 4.3 数据模型 (Core Data)

#### Note Entity
```
Note
├── id: UUID
├── title: String
├── content: String?
├── status: Int16 (0=待办, 1=进行中, 2=已完成)
├── priority: Int16 (0=低, 1=中, 2=高, 3=紧急)
├── tags: String? (JSON 数组)
├── reminderDate: Date?
├── repeatType: Int16? (0=不重复, 1=每天, 2=每周, 3=每月)
├── sortOrder: Int32
├── createdAt: Date
└── updatedAt: Date
```

#### SearchRecord Entity
```
SearchRecord
├── id: UUID
├── query: String
├── type: Int16 (0=术语, 1=代码, 2=问答, 3=总结)
├── result: String
├── provider: String
├── createdAt: Date
```

#### TranslationRecord Entity
```
TranslationRecord
├── id: UUID
├── sourceText: String
├── targetText: String
├── sourceLanguage: String
├── targetLanguage: String
├── isFavorite: Bool
├── createdAt: Date
```

## 5. 开发计划

### 5.1 阶段划分

| 阶段 | 名称 | 主要内容 | 预估工作量 |
|-----|------|---------|-----------|
| P1 | 基础架构 | 项目搭建 + 悬浮窗口 + 菜单栏 + 角色动画 | 基础 |
| P2 | 便签系统 | 便签 CRUD + 标签 + 优先级 + 本地提醒 | 核心 |
| P3 | LLM 集成 | 多提供商适配 + 流式输出 + 配置管理 | 核心 |
| P4 | 智能搜索 | 搜索界面 + AI 调用 + 历史记录 | 中等 |
| P5 | 翻译功能 | 翻译界面 + 多语言 + 历史收藏 | 中等 |
| P6 | 优化完善 | UI 美化 + 性能优化 + Bug 修复 | 收尾 |
